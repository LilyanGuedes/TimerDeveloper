#  TimerDevelopment

Este projeto refere-se a um temporizador, efectua a contagem decrescente a partir de um valor predeterminado e alerta-o quando chegar a zero. Você poderá também salvar padrões de tempo.

## 🛠️ Construído com

HTML, CSS, JAVASCRIPT
